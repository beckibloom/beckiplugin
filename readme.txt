== Becki's Plugin ==
Author: beckibloom

1. Creates a basic custom post type "Reviews" which is included among main posts on main landing page.
2. Creates shortcode [user_ip_address] which can be used to display the current user's IP address. 
  - IP address data is stored as transient for one hour and refreshes as needed.
  - IP address data is retrieved from http://bot.whatismyipaddress.com/
